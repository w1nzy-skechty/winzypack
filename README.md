# WinzyPack - Ultimate Gaming Optimizer

WinzyPack is an open-source collection of performance optimization scripts designed to help PC enthusiasts squeeze every drop of power out of their hardware. 

## 🛡️ Safety & Transparency
We believe in 100% transparency. Our tools are not "black boxes"; they are simple, human-readable PowerShell (`.ps1`) and Batch (`.bat`) scripts.

### Why do Antiviruses flag my files?
If you scan our `.exe` or `.bat` files on **VirusTotal**, you might see a few "Generic" detection flags (e.g., 12/70). **This is a 100% False Positive.**

*   **Registry Changes:** Windows security software often panics when an unsigned script attempts to modify Registry keys or Windows Defender settings. This is exactly what optimization tools do to improve performance.
*   **Unsigned Code:** These scripts are compiled by us for your convenience. Since they lack a paid Microsoft Digital Certificate, some AI-based antivirus scanners flag them automatically as "suspicious" simply because they are not digitally signed.
*   **Verification:** You don't have to take our word for it. You can **Right-Click** any of our files and select **"Edit"** (for .bat) or open them in **Notepad** (for .ps1). You will see the plain text code, which confirms they only perform harmless system tweaks.

## ⚠️ Best Practices
While our scripts are designed to be safe, modifying Windows system settings carries inherent risks. We recommend:

1.  **Create a Restore Point:** Before running any optimization pack, always go to Windows Search and type "Create a restore point". This ensures you can revert your system to its previous state if anything goes wrong.
2.  **Run as Administrator:** Our scripts require Administrative privileges to modify system services and registry entries.
3.  **Restart Your PC:** Most changes (especially those related to Windows services) require a system restart to take effect fully.
